Workout Beat Support

Official support page for Workout Beat / 節拍運動 / 节拍运动

English

If you encounter a problem, have a suggestion, or want to report a bug, please use the support repository issue tracker.

Open a support request

  • Please include your iPhone model, iOS version, and a short description of what happened.
  • If the issue involves beat playback, timer behavior, voice cues, or flashlight sync, mention which setting you were using.

繁體中文

如果你遇到問題、想提供建議，或要回報 bug，請使用這個支援 repo 的 issue 頁面。

提出支援需求

  • 建議附上你的 iPhone 型號、iOS 版本，以及問題發生時的簡短描述。
  • 如果問題和節拍播放、倒數計時、語音口令或閃光燈同步有關，也可以一起寫上當時使用的設定。

简体中文

如果你遇到问题、想提出建议，或要反馈 bug，请使用这个支持 repo 的 issue 页面。

提交支持请求

  • 建议附上你的 iPhone 型号、iOS 版本，以及问题发生时的简短描述。
  • 如果问题和节拍播放、倒计时、语音口令或闪光灯同步有关，也可以一并写上当时使用的设置。
